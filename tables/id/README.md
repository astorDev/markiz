# Table Ids

Having an id for a markdown table is essential to make a use of markdown table as a spreadsheet-like source possible: For example when a table represent some financial or statistical data. Unlike in standard markdown with introduction of [sections](../../sections/README.md) we have a rather unabimbigous way to generate an id for a table: `{section-name}-t{table order number}`. In a rare case, when a table is out of section (for example when you just have an untitled markdown with a table) the id will be simply that: `t{table order number}`
